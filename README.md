# crawler
Website crawler, Python 

## Installation

Check out from Github

Install dependencies/modules

```  pip install beautifulsoup4 ``` 
or (if like me you have Python 2.x and Python 3.x)
```  pip3 install bs4 ``` 
```  pip3 install requests ``` 
```  pip3 install lxml ``` 

Execution
```python3 crawler.py```
